# ①課題名
キャンプ場記録アプリ

## ②課題内容（どんな作品か）
- 過去に行ったことのあるキャンプ場を記録可能
- キャンプ場名と都道府県を入力するだけで、自動でGoogleマップ上でのリンクが発行される

## ③アプリのデプロイURL
デプロイしている場合はURLを記入（任意）

## ④アプリのログイン用IDまたはPassword（ある場合）
- なし

## ⑤工夫した点・こだわった点
- localStorageへ複数の要素を格納するために、JSONへの変換方法を工夫した
- UIでリストよりも表形式が見やすいと思い、localStorageのデータを表形式へ出力させるための設定に手こずった

## ⑥難しかった点・次回トライしたいこと（又は機能）
- 位置情報の扱いが難しかった（今回はかなり力技）
- 登録したキャンプ場の位置情報をもとに、日本地図にフラッグをたてられるようにしたい
